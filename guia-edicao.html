Guia de Edição - Landing Page

Este guia explica como editar facilmente sua landing page sem precisar de conhecimentos avançados de programação.

Estrutura de Arquivos

  • index.html - Arquivo principal com todo o conteúdo da página
  • css/styles.css - Estilos visuais (cores, tamanhos, espaçamentos)
  • js/main.js - Funcionalidades interativas (slider, formulário, menu mobile)
  • assets/ - Pasta com todas as imagens

Como Editar Textos

Para alterar qualquer texto da landing page:

  1. Abra o arquivo index.html em qualquer editor de texto (Notepad, VS Code, etc.)
  2. Localize o texto que deseja alterar
  3. Substitua o texto mantendo as tags HTML que o envolvem
Dica: Use a função de busca (Ctrl+F) para encontrar rapidamente o texto que deseja editar.

Exemplo:

Para alterar o título principal "TRANSFORME SUA IDEIA EM UM PRODUTO DIGITAL REAL", localize:

<h1>TRANSFORME SUA IDEIA EM UM PRODUTO DIGITAL REAL</h1>

E substitua por:

<h1>SEU NOVO TÍTULO AQUI</h1>

Como Alterar Imagens

Você pode substituir as imagens de duas maneiras:

Método 1: Substituir os arquivos (mais fácil)

  1. Prepare sua nova imagem com o mesmo nome do arquivo original
  2. Substitua o arquivo na pasta assets/ ou assets/testimonials/

Método 2: Alterar o caminho no HTML

  1. Adicione sua nova imagem à pasta assets/
  2. No arquivo index.html, localize a tag <img> da imagem que deseja substituir
  3. Altere o atributo src para apontar para sua nova imagem

Exemplo:

<img src="assets/minhafoto.jpg" alt="Alisson Rosa - Desenvolvedor">

Altere para:

<img src="assets/nova-foto.jpg" alt="Alisson Rosa - Desenvolvedor">

Como Alterar Cores

Todas as cores principais estão definidas como variáveis no início do arquivo CSS:

  1. Abra o arquivo css/styles.css
  2. Localize a seção :root no início do arquivo
  3. Altere os valores das variáveis para suas cores desejadas

Exemplo:

:root {
    --accent: #9747FF; /* Cor principal/destaque */
    --bg-dark: #0F0F12; /* Cor de fundo escura */
    /* outras variáveis... */
}
Dica: Use um seletor de cores online como HTML Color Codes para encontrar códigos de cores.

Como Editar o Formulário

Para modificar os campos do formulário:

  1. Localize a seção <section id="formulario-contato"> no arquivo index.html
  2. Dentro da tag <form>, você pode:
    • Adicionar novos campos copiando e modificando os existentes
    • Remover campos que não deseja
    • Alterar os textos dos rótulos (labels)
    • Modificar as opções do dropdown
Importante: Se você estiver integrando com um serviço de envio de formulário real, precisará editar também o arquivo js/main.js na seção "4. FORMULÁRIO DE CONTATO".

Como Editar os Depoimentos

Para adicionar, remover ou modificar depoimentos:

  1. Localize a seção <section id="depoimentos"> no arquivo index.html
  2. Cada depoimento está dentro de um elemento <div class="testimonial-slide">
  3. Para adicionar um novo depoimento, duplique um dos blocos existentes e altere o conteúdo
  4. Para remover um depoimento, exclua o bloco <div class="testimonial-slide"> correspondente
  5. Lembre-se de atualizar os "dots" de navegação para corresponder ao número de depoimentos

Publicação

Para publicar sua landing page:

  1. Faça upload de todos os arquivos e pastas para seu servidor web ou serviço de hospedagem
  2. Mantenha a mesma estrutura de pastas
  3. O arquivo index.html deve estar na raiz
Dica: Serviços como GitHub Pages, Netlify ou Vercel permitem publicar sites estáticos gratuitamente e com facilidade.
